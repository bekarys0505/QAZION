HTML Формалар

HTML формалар пайдаланушыдан мәліметтер жинауға арналған. Формалар <form> тегінде жасалады және түрлі input түрлерін қамтиды.

Мысал:


<form class="form-example">
  <label>Аты-жөні:</label>
  <input type="text" placeholder="Атыңызды енгізіңіз">

  <label>Электрондық пошта:</label>
  <input type="email" placeholder="email@domain.com">

  <label>Пароль:</label>
  <input type="password" placeholder="Пароль енгізіңіз">

  <label>Жыныс:</label>
  <select>
    <option>Ер</option>
    <option>Әйел</option>
  </select>

  <label>Пікіріңіз:</label>
  <textarea placeholder="Мәтінді енгізіңіз"></textarea>

  <button type="submit">Жіберу</button>
</form>
