CSS Box Model

Box Model — әрбір HTML элементінің құрылымын көрсетеді: content, padding, border, margin.

Мысал коды:

<div class='box'>Бұл box model мысалы</div>
.box {
  width: 200px;
  padding: 20px;
  border: 5px solid #00759c;
  margin: 30px;
  background-color: #6dd5fa;
  text-align: center;
}
