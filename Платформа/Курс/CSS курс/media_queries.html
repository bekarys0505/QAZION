CSS Media Queries

Media Queries арқылы экран өлшеміне байланысты стильдерді өзгертуге болады. Мысалы, мобильдік экран үшін блоктардың түсін өзгертеміз.

Мысал:


<div class="box">Блок 1</div>
<div class="box">Блок 2</div>

<style>
.box {
  width: 200px;
  height: 100px;
  background-color: #6dd5fa;
  margin: 10px;
  display: flex;
  align-items: center;
  justify-content: center;
  color: #000;
  font-weight: bold;
  border-radius: 10px;
}

/* Мобильдік экрандар үшін */
@media (max-width: 600px) {
  .box {
    background-color: #ff7f50;
    width: 100%;
  }
}
</style>
