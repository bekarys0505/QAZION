CSS Мәтінді стильдеу

CSS арқылы мәтіннің түрін, өлшемін, түсін және басқа қасиеттерін өзгертуге болады:

  • Шрифт — font-family қасиеті арқылы өзгереді. Мысалы: p { font-family: Arial; }
  • Өлшем — font-size қасиеті арқылы мәтін өлшемін орнатамыз. Мысалы: h1 { font-size: 36px; }
  • Түсі — color қасиеті арқылы мәтін түсін белгілейміз. Мысалы: span { color: red; }
  • Жуандық, курсив, астын сызу — font-weight, font-style, text-decoration арқылы өзгертеміз.
  • Түсі мен фонды қосу — color және background-color бір уақытта қолдануға болады.

Мысал:

<!-- Шрифт пен өлшем -->
p {
  font-family: 'Verdana';
  font-size: 18px;
}

<!-- Жуан және курсив мәтін -->
span {
  font-weight: bold;
  font-style: italic;
}

<!-- Мәтін түсі және фон -->
div {
  color: white;
  background-color: red;
  padding: 10px;
}
