CSS Позициялау және Flex/Grid

1️⃣ Position мысалы

Relative және Absolute позициялау. Relative өз орнында сәл жылжиды, Absolute ата-анасына қатысты дәл орналасады.


<div class="position-box relative">Relative</div>
<div class="absolute-parent">
  <div class="absolute-child">Absolute</div>
</div>

2️⃣ Flexbox мысалы

Flexbox блоктарды қатармен орналастырады. Gap арқылы аралары ашылады.


<div class="flex-container">
  <div class="position-box">Box 1</div>
  <div class="position-box">Box 2</div>
  <div class="position-box">Box 3</div>
</div>

3️⃣ Grid мысалы

Grid блоктарды тор сияқты орналастырады: 3 баған × бірнеше қатар.


<div class="grid-container">
  <div class="position-box">1</div>
  <div class="position-box">2</div>
  <div class="position-box">3</div>
  <div class="position-box">4</div>
  <div class="position-box">5</div>
  <div class="position-box">6</div>
</div>
