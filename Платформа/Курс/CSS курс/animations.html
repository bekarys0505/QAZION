CSS Animations

CSS арқылы элементтерді қозғалысқа келтіруге болады. Бұл @keyframes арқылы жасалады.

Автоматты анимация мысалы:

<div class="animate-box">Move</div>

@keyframes moveBox {
  0% { transform: translateY(0) rotate(0deg); background-color:#ff7f50; }
  50% { transform: translateY(-50px) rotate(180deg); background-color:#6dd5fa; }
  100% { transform: translateY(0) rotate(360deg); background-color:#ffb347; }
}

.animate-box {
  animation: moveBox 3s infinite alternate;
}

Hover анимация мысалы:

<div class="hover-box">Hover</div>

.hover-box {
  transition: transform 0.5s, background-color 0.5s;
}
.hover-box:hover {
  transform: scale(1.5) rotate(15deg);
  background-color: #3a8dff;
}
