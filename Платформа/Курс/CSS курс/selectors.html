CSS Селекторлар

CSS селекторлары арқылы HTML элементтерін таңдап, оларға стиль қолданамыз:

  • Тег селекторы — барлық сол тегтегі элементтерге әсер етеді. Мысалы: p { color: red; }
  • Класс селекторы — белгілі бір класқа әсер етеді. Мысалы: .highlight { background: yellow; }
  • ID селекторы — белгілі бір идентификаторға әсер етеді. Мысалы: #main-title { font-size: 36px; }
  • Түрлі селекторларды біріктіру — div, p { margin: 10px; }

Мысал:

<p>Бұл тег селекторы арқылы қызыл түсті мәтін</p>
<p class='highlight'>Бұл класс селекторы арқылы сары фонды мәтін</p>
<h1 id='main-title'>Бұл ID селекторы арқылы үлкен және жуан мәтін</h1>
