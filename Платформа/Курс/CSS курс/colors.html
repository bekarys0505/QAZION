CSS Түстер мен фондар

CSS арқылы веб-беттерге түстер мен фон қолдануға болады. Түстерді color қасиеті арқылы мәтінге, background-color арқылы фонға қолданамыз.

  • Мәтін түсі — color қасиеті арқылы өзгереді.
  • Фон түсі — background-color қасиеті арқылы өзгереді.
  • Градиенттер — linear-gradient немесе radial-gradient арқылы стильді фон жасауға болады.

Мысал:

<p style='color:blue;'>Көк мәтін</p>
<div style='background-color:yellow; padding:10px;'>Сары фон</div>
<section style='width:100%;height:50px;background:linear-gradient(45deg,#ff0000,#0000ff);'></section>
