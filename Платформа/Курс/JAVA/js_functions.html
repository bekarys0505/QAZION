Функциялар

Функция — белгілі бір операцияларды орындайтын код блоктары. Функциялар кодты қайта қолдануға мүмкіндік береді және бағдарламаны жеңілдетеді.

1. Функцияны анықтау

function функцияАты(параметр1, параметр2) {
  // функция денесі
  return мән;
}

Мысал:

<script>
function қосу(a, b) {
  return a + b;
}

let нәтиже = қосу(5, 3);
console.log("Нәтиже: " + нәтиже);
</script>

Бұл мысалда қосу функциясы екі санды қосып, нәтижесін қайтарады.

2. Функцияны шақыру

<script>
function сәлемдесу(аты) {
  alert("Сәлем, " + аты + "!");
}

сәлемдесу("Бекарыс");
</script>
