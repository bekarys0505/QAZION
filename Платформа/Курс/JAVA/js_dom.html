DOM және оқиғалар

DOM (Document Object Model) — HTML құжаттың құрылымын объектілер түрінде көрсету. JavaScript арқылы осы объектілерді өзгертуге, қосымша интерактивтілік жасауға болады.

1. Элементті таңдау

// ID арқылы
let title = document.getElementById("title");

// Класс арқылы
let items = document.getElementsByClassName("item");

// Тег арқылы
let paragraphs = document.getElementsByTagName("p");

// querySelector арқылы (бірінші табылған элемент)
let firstItem = document.querySelector(".item");

// querySelectorAll (барлық табылған элементтер)
let allItems = document.querySelectorAll(".item");

2. Элементке мән беру

title.textContent = "Жаңа тақырып";
title.style.color = "blue";
title.style.fontSize = "24px";

3. Оқиғалар қосу

let button = document.getElementById("btn");

button.addEventListener("click", function() {
  alert("Сіз батырманы бастыңыз!");
});

4. Мысал интерактивті

<h2 id="title">QAZION</h2>
<button id="btn">Бастыру</button>

<script>
let title = document.getElementById("title");
let button = document.getElementById("btn");

button.addEventListener("click", function() {
  title.textContent = "Сәлем, оқушы!";
  title.style.color = "green";
});
</script>
