Объектілер

Объект — қасиеттер мен әдістерді сақтауға арналған құрылым. Қасиеттер (properties) мәндерді сақтайды, әдістер (methods) — функциялар.

1. Объектіні анықтау

let person = {
  name: "Бекарыс",
  age: 19,
  city: "Алматы"
};

console.log(person.name); // "Бекарыс"

2. Қасиет қосу және өзгерту

person.job = "Студент"; // жаңа қасиет қосу
person.age = 20; // бар қасиетті өзгерту

3. Әдістер (methods) қосу

let person = {
  name: "Бекарыс",
  greet: function() {
    return "Сәлем, мен " + this.name;
  }
};

console.log(person.greet()); // "Сәлем, мен Бекарыс"

4. Цикл арқылы объектінің қасиеттерін оқу

for (let key in person) {
  console.log(key + ": " + person[key]);
}
