Айнымалылар және мәліметтер түрлері

JavaScript-те айнымалылар – деректерді сақтау үшін қолданылады. Айнымалыға мән беру арқылы біз деректерді сақтай аламыз және оларды әртүрлі есептеулерде қолдана аламыз.

Айнымалы жариялау

  • let – өзгеретін мәндер үшін қолданылады.
  • const – тұрақты мәндер үшін қолданылады.
  • var – ескі синтаксис, қазіргі кезде аз қолданылады.

Мәліметтер түрлері

  • Number – сандар: бүтін және бөлшек (мысалы: 10, 3.14)
  • String – мәтін: сөздер немесе сөйлемдер (мысалы: "Сәлем")
  • Boolean – логикалық мән: true немесе false
  • Array – мәндер тізімі (мысалы: [1,2,3])
  • Object – кілт-мән жұптары (мысалы: {name: "Ali", age: 20})
  • Null және Undefined – бос немесе анықталмаған мән

Мысал:

<script>
let name = "Ali";
const age = 20;
let isStudent = true;

let fruits = ["Алма", "Банан", "Шие"];
let person = {name: "Ali", age: 20};

console.log("Аты: " + name);
console.log("Жасы: " + age);
console.log("Студент: " + isStudent);
console.log("Жемістер: " + fruits);
console.log("Адам: ", person);
</script>
