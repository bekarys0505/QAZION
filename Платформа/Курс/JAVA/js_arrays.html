Массивтер

Массив — бір типтегі немесе әртүрлі типтегі элементтерді сақтауға арналған құрылым. Массив элементтері индекспен шақырылады, индекстер 0-ден басталады.

1. Массивті анықтау

let fruits = ["Алма", "Банан", "Шие"];
console.log(fruits[0]); // "Алма"

2. Массивке элемент қосу

fruits.push("Апельсин"); // соңына қосу
fruits.unshift("Қарақат"); // басына қосу

3. Массивтен элемент алу

fruits.pop(); // соңғы элементті жою
fruits.shift(); // бірінші элементті жою

4. Массивті цикл арқылы көрсету

for(let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

Мысал:

<script>
let fruits = ["Алма", "Банан", "Шие"];
fruits.push("Апельсин");

</script>
