Шарттар (if/else)

JavaScript-де шартты операторлар if, else if және else арқылы логикалық тексерулер жасауға мүмкіндік береді. Бұл операторлар белгілі бір шарт дұрыс болса, белгілі бір код блогын орындауға қолданылады.

Синтаксис:

if (шарт) {
  // шарт дұрыс болса орындалатын код
} else if (басқаШарт) {
  // басқа шарт дұрыс болса орындалатын код
} else {
  // барлық шарттар дұрыс болмаса орындалатын код
}

Мысал 1: if/else

<script>
let age = 18;

if (age >= 18) {
  console.log("Сіз ересексіз");
} else {
  console.log("Сіз әлі кәмелет жасқа толмағансыз");
}
</script>

Мысал 2: if/else if/else

<script>
let score = 75;

if (score >= 90) {
  console.log("A деңгейі");
} else if (score >= 75) {
  console.log("B деңгейі");
} else if (score >= 50) {
  console.log("C деңгейі");
} else {
  console.log("Өтпеді");
}
</script>
