JavaScript Негіздері

JavaScript — бұл веб-беттерді интерактивті етуге арналған бағдарламалау тілі. JS көмегімен біз түймелерді басқанда әрекет жасау, форма мәндерін тексеру, динамикалық мазмұн қосу сияқты функцияларды жүзеге асыра аламыз.

JS-ті HTML бетінде қолдану:

JavaScript кодын қосу үшін бірнеше әдіс бар:

  • Inline (тікелей HTML тегінде): onclick="..." сияқты атрибуттар арқылы.
  • Internal (ішкі скрипт): <script>...</script> тегін <body> немесе <head> ішінде қолдану.
  • External (сыртқы файл): <script src="script.js"></script> арқылы сыртқы JS файлын қосу.

Мысал:

Түймеге басқанда хабарлама шығаратын қарапайым JavaScript:

<!DOCTYPE html>
<html>
<head>
  <title>JS Негіздері</title>
</head>
<body>

  <h2>Қарапайым батырма</h2>
  <button onclick="sayHello()">Басыңыз</button>

  <script>
    function sayHello() {
      alert("Сәлем, QAZION оқушысы!");
    }
  </script>

</body>
</html>
