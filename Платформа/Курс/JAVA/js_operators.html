Операторлар және есептеулер

JavaScript-те есептеулер үшін әртүрлі операторлар қолданылады. Операторлар айнымалылар мен мәндерге амалдар жасауға мүмкіндік береді.

Сандық операторлар

  • + – қосу
  • - – алу
  • * – көбейту
  • / – бөлу
  • % – қалдық
  • ++ – бірлікке көбейту (инкремент)
  • -- – бірлікке азайту (декремент)

Мәліметтерді салыстыру операторлары

  • == – тең бе
  • === – мәні мен типі тең бе
  • != – тең емес
  • !== – мәні немесе типі тең емес
  • >, <, >=, <= – салыстыру

Логикалық операторлар

  • && – және
  • || – немесе
  • ! – теріс

Мысал:

<script>
let a = 10;
let b = 5;

// Арифметикалық операциялар
let sum = a + b;
let diff = a - b;
let mult = a * b;
let div = a / b;
let mod = a % b;

// Салыстыру
let isEqual = a === b;
let isGreater = a > b;

// Логикалық
let check = (a > b) && (b < 10);

console.log("Қосу: " + sum);
console.log("Алу: " + diff);
console.log("Көбейту: " + mult);
console.log("Бөлу: " + div);
console.log("Қалдық: " + mod);
console.log("Тең бе: " + isEqual);
console.log("Үлкен бе: " + isGreater);
console.log("Логикалық тексеру: " + check);
</script>
