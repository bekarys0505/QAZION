Циклдар (for, while)

JavaScript-де циклдар бірдей операцияларды бірнеше рет орындау үшін қолданылады. Ең көп қолданылатын циклдар — for және while.

1. For циклі

for циклі белгілі бір санға дейін қайталанатын операцияларды орындауға арналған.

for (бастау; шарт; қадам) {
  // орындалатын код
}

Мысал:

<script>
for (let i = 1; i <= 5; i++) {
  console.log("Сан: " + i);
}
</script>

2. While циклі

while циклі шарт дұрыс болғанша кодты қайталайды.

<script>
let i = 1;
while (i <= 5) {
  console.log("Сан: " + i);
  i++;
}
</script>
