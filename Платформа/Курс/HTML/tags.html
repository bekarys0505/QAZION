Негізгі тегтер

HTML құжатында қолданылатын негізгі тегтер:

  • <html> — құжат HTML тілінде екенін көрсетеді
  • <head> — метадеректерді қамтиды
  • <title> — бет тақырыбы
  • <body> — көрінетін контент

Мысал:

<!DOCTYPE html>
<html>
<head>
  <title>Менің бетім</title>
</head>
<body>
  <h1>Сәлем, әлем!</h1>
</body>
</html>
